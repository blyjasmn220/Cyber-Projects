# Cyber Projects & Security Toolkit 

## Overview 
A collection of documentation, scripts, and research files focused on network security, penetration testing, and malware analysis.  

## Repository Contents 
* **Firewalls/**: Detailed guides on configuring and testing network defenses. 
* **Penetration Devices/**: Research on hardware tools like ESP32 S2, Pwngotchi, and Flipper Zero. 
* **Malware Research/**: Analysis of malicious activities performed via Windows CMD and Python.  

## SKills & Tools
* **Networking:** TCP/UDP protocols, Firewalls configuration. 
* **Security:** Evil Tein attacks, Router access methodologies. 
* **Scripting:** Python-based cyber toolkits.
